# Brawl Stars Monthly Finals 2025 Music Player

A React application designed as a customizable track and trigger pad, specifically built to play and manage production music from the Brawl Stars Monthly Finals 2025. 

- NOTE: This app was completely coded by Antigravity AI, and ideated/prompted by me, Aga2522.

## Features
- Interactive soundboard and music player.
- Easy to use interface for triggering tracks during live events or personal use.
- Built with React and Vite for fast performance.

## Playing Locally

To run this project locally, follow these steps:

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd bs-mf-2025-music-player
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```
